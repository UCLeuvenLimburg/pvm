\begin{frame}
  \frametitle{Generics (Java/\csharp)}
  \begin{itemize}
    \item Java and \csharp\ support generic classes
    \item Classes parameterised in one or more type variables
    \item Typical use: container classes
          \begin{itemize}
            \item \texttt{List<T>}
            \item \texttt{Set<T>}
            \item \texttt{Map<K, V>}
            \item \dots
          \end{itemize}
    \item You can define your own
  \end{itemize}
\end{frame}
